Controls (Keyboard):
			start menu:
						Q: Quit the Game
						Enter: Start the game
						F1: Mute/Unmute the Audio
						F2: Volume Down
						F3: Volume Up

			paused:
						Q: Quit the Game
						left/right shift: UnPause the game
						F1: Mute/Unmute the Audio
						F2: Volume Down
						F3: Volume Up

			In Game:
						Q: Quit the Game
						left/right shift: Pause the game
						F1: Mute/Unmute the Audio
						F2: Volume Down
						F3: Volume Up

						WASD/Arrow Keys: moves the player
						Z/N: Attack
						X/M: Use Item
						Escape: Open HUD
			
			HUD:
						A/D/Arrows: Select Item
						Escape: Close HUD
						Q: Quit Game

Controls (Gamepad):

			Start Menu:
						Back: Quit the game
						Start: Start the game

			Paused:
						Back: Quit the game
						BigButton: Start the Game

			In Game:
			
						Back: Quit the game
						Start: Enter HUD
						BigButton: Puase the Game
						DPAD/Left Stick: Move the character

						A: Attack
						B: Use item
